# priceLevels module — database schema

Source: `server/priceLevels/priceLevel.js`
SQLite table: `price_levels` -> Postgres table: `price_levels`

## Table: `price_levels`

Stores one row per (company, ordinal price-level slot). `level_index` is the
zero-based position; `name` is the display label for that slot.

| Column | Postgres type | Nullable | Default | Notes |
|---|---|---|---|---|
| `price_level_id` | `BIGINT GENERATED BY DEFAULT AS IDENTITY` | NO | identity | Primary key. SQLite `INTEGER PRIMARY KEY AUTOINCREMENT`. |
| `company_id` | `BIGINT` | NO | — | Owning company. **Explicit FK** -> `companies(company_id)` `ON DELETE CASCADE`. |
| `level_index` | `INTEGER` | NO | — | Zero-based ordinal slot within the company. |
| `name` | `TEXT` | NO | `''` | Price-level display name. SQLite `TEXT NOT NULL DEFAULT ''`. |
| `is_active` | `BOOLEAN` | YES | `TRUE` | SQLite `INTEGER` 0/1 flag (default 1). 0 -> false, 1 -> true. Not read/written by the service. |
| `created_at` | `TIMESTAMPTZ` | YES | `now()` | SQLite `TEXT` ISO-8601 via `datetime('now')`. App stores ISO strings. |
| `updated_at` | `TIMESTAMPTZ` | YES | `now()` | SQLite `TEXT` ISO-8601 via `datetime('now')`. Set to `datetime('now')` on each insert. |

### Constraints

- **Primary key:** `price_level_id`.
- **Unique:** `(company_id, level_index)` — one price level per slot per company.
- **Foreign key (explicit in source):** `company_id` -> `companies(company_id)`,
  `ON DELETE CASCADE`.

### Indexes

- `idx_price_levels_company_id` on `(company_id)` — added for FK lookup/join
  performance.

## Relationships

- `price_levels.company_id` -> `companies.company_id` — **explicit** FK in the
  SQLite source (`REFERENCES companies(company_id) ON DELETE CASCADE`). Deleting
  a company removes all of its price levels.

## Notes

- **No money/quantity columns** exist in this table, so no `NUMERIC(18,x)`
  mappings apply here.
- The service treats the table as a per-company list: `save` deletes all rows
  for the company and re-inserts them in array order, keeping `level_index`
  contiguous from 0. `is_active` is never updated and stays at its default.
