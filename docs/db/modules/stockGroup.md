# stockGroup — database schema

Source: `server/stockGroup/stockGroup.js` (SQLite). Postgres translation in
`docs/db/modules/stockGroup.sql`.

This module defines a single table: **`stock_groups`** — the inventory
stock-group hierarchy (groups and subgroups) scoped per company.

## Table: `stock_groups`

| Column | Postgres type | Nullable | Default | Notes |
|---|---|---|---|---|
| `sg_id` | `BIGINT GENERATED BY DEFAULT AS IDENTITY` | no | identity | Primary key. SQLite `INTEGER PRIMARY KEY AUTOINCREMENT`. |
| `company_id` | `BIGINT` | no | — | Owning company. **Explicit FK** -> `companies(company_id)` `ON DELETE CASCADE`. |
| `name` | `TEXT` | no | — | Stock-group name. |
| `alias` | `TEXT` | yes | — | Optional alias. |
| `parent_group_id` | `BIGINT` | yes | — | Self-reference for hierarchy. **Explicit FK** -> `stock_groups(sg_id)`. NULL = top-level. |
| `should_quantities_be_added` | `BOOLEAN` | no | `false` | SQLite `INTEGER DEFAULT 0`; 0->false, 1->true. |
| `hsn_sac_code` | `TEXT` | yes | — | HSN/SAC tax classification code. |
| `hsn_sac_description` | `TEXT` | yes | — | HSN/SAC description. |
| `gst_rate` | `NUMERIC(18,4)` | no | `0` | GST percentage rate. Quantity/rate — never floating point. |
| `cgst_rate` | `NUMERIC(18,4)` | no | `0` | CGST percentage rate. Quantity/rate — never floating point. |
| `sgst_rate` | `NUMERIC(18,4)` | no | `0` | SGST percentage rate. Quantity/rate — never floating point. |
| `taxability_type` | `TEXT` | yes | `NULL` | Taxability classification. Added via post-create `ALTER TABLE` in source (idempotent). |
| `statutory_details` | `TEXT` | yes | — | Free-form statutory details. |
| `is_primary` | `BOOLEAN` | no | `false` | SQLite `INTEGER DEFAULT 0`; 0->false, 1->true. |
| `is_active` | `BOOLEAN` | no | `true` | SQLite `INTEGER DEFAULT 1`; 0->false, 1->true. Soft-delete marker (delete sets it false). |
| `is_predefined` | `BOOLEAN` | no | `false` | SQLite `INTEGER DEFAULT 0`; 0->false, 1->true. Predefined groups cannot be edited or deleted. |
| `created_at` | `TIMESTAMPTZ` | no | `now()` | SQLite `TEXT DEFAULT (datetime('now'))`; ISO-8601 string. |
| `updated_at` | `TIMESTAMPTZ` | no | `now()` | SQLite `TEXT DEFAULT (datetime('now'))`; ISO-8601 string. Touched on update. |

### Constraints & indexes

- Partial unique index `uq_stock_groups_company_lower_name_active` on
  `(company_id, LOWER(name)) WHERE is_active = true`. **Inferred** from the
  service `create()` duplicate check; there is no DB-level UNIQUE in the SQLite
  source.
- Indexes on FK columns: `company_id`, `parent_group_id`.

## Relationships

- `stock_groups.company_id` -> `companies.company_id` (**explicit** in source,
  `ON DELETE CASCADE`). Many stock groups per company.
- `stock_groups.parent_group_id` -> `stock_groups.sg_id` (**explicit** self-FK in
  source). Builds the group/subgroup tree; NULL marks top-level groups.

## Money / quantity columns

The `*_rate` columns (`gst_rate`, `cgst_rate`, `sgst_rate`) hold tax percentage
rates and are mapped to `NUMERIC(18,4)`. They must never use floating-point
types, to avoid rounding errors in tax computation.
